## üë• Role-Based Features

### 1. Patient Features
*   **Dashboard**: Overview of vitals, upcoming tasks, and rewards.
*   **Health Profile**: Manage personal data (allergies, conditions, vitals).
*   **Appointment Booking**: Search for experts and schedule consultations.
*   **Dietary & Goal Tracking**: Interactive checklists for daily health tasks.

### 2. Expert (Doctor) Features
*   **Expert Dashboard**: View daily schedule and appointment requests.
*   **Patient Management**: Access permitted patient profiles and history.
*   **Verification System**: Upload professional credentials for verification.
*   **Schedule Management**: Set availability for appointments.

### 3. Admin Features
*   **User Management**: View user lists, manage account status (suspend/active).
*   **Expert Verification**: Review and approve expert documents.
*   **Audit Trails**: Monitor system-wide sensitive actions for compliance.
*   **Analytics**: View platform stats (user growth, token distribution).
